About Paradax

The concept was first used at Peecho, the cloud print provider.
At first, the reason was very mundane: we thought it was better not to let our Java developers fiddle with the front-end code too much and only let them generate structured content instead. That makes much more sense to them, anyway.

The Paradax website you are looking at is a simplified example of this technique. It was created as a weekend project at Peecho. Read more on the Peecho blog.
