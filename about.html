About Paradax

The concept was first used at Peecho, the cloud print provider.
At first, the reason was very mundane: we thought it was more efficient not to let our Java developers fiddle with the front-end code that much. Structured data just makes much more sense to them, anyway. That said, the SEO and speed effects are awesome!

The Paradax website was created as a weekend project at Peecho. Read more on the Peecho blog.
